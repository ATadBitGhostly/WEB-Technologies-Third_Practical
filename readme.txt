--------- EN ---------

To do task 1:

1. Download XAMPP (you probably already have it) activate both Apache and MySQL
2. Click on the Admin button located on the MySQL row and you will be taken to the phpmyadmin dashboard
3. Create a new db named "webdev_project".
4. Open setup.php via this path "http://localhost/WEB-Technologies-Third_Practical/includes/setup.php" to create the tables
